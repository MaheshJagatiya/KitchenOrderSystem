Unity Cooking Game Prototype

Overview
This is a time-based cooking game prototype built using Unity and C#.
The player prepares and delivers food items to multiple customer windows within a limited time.

Gameplay

Pick ingredients from the refrigerator
Chop vegetables on the table
Cook meat on the stove
Deliver items to the correct window
Complete orders to earn score
Manage time efficiently

Features

Multiple order windows
Ingredient system (Vegetable, Meat, Cheese)
Preparation system (Chopping and Cooking)
Order validation system
Timer-based gameplay
Score and High Score system
Floating text feedback
Pause and Resume functionality
Event-driven UI

Architecture
This project follows MVC pattern

Model
OrderData
IngredientData

View
PlayerView
CustomerWindowView
UI Views

Controller
GameController
OrderController
ScoreController

Unity Version - 6.0(6000.0.61)

Controls
Move - WASD or Arrow Keys
Interact - E
Pause - UI Button
